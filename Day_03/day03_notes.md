Boolean values:
- True
- False
(First letters must be capital)

Assignment operator:
= assigns a value to a variable

Arithmetic operators:
+ addition
- subtraction
* multiplication
/ division
% modulus (remainder)
// floor division (rounds down)
** exponent (power)

Data types reviewed:
- float (decimal numbers)
- complex numbers (example: 1 + 1j)

Comparison operators:
> greater than
< less than
>= greater or equal
<= less or equal
== equal
!= not equal

Identity operators:
is: checks if two things are the same object
is not: checks if they are not the same object

Membership operators:
in: checks if something exists in a string/list/etc
not in: checks if something does NOT exist

Logical operators:
and: both conditions must be true
or: at least one condition is true
not: flips True/False

Variables are used to store values (example: name = "Karl")

Use len() to find how many characters are in a string

Use type() to check the datatype

Datatypes covered today:
- str: text inside quotes
- int: whole numbers
- float: numbers with decimals
- bool: True or False values

Operators practiced:
- + addition
- - subtraction
- * multiplication
- / division
- % modulus (remainder)
- ** exponent
- // floor division (rounds down)

input() is used to take user input
input() always returns a string

If doing math with input, convert it:
- int() converts to integer
- float() converts to decimal number

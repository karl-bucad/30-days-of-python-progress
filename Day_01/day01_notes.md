Use triple quotes (""") for multiline comments

Variables store information (example: x = 5)

Basic datatypes:
- int: whole numbers
- float: decimal numbers
- str: text in quotes

Use print() to display output

Use type() to find the datatype of a value

Operators:
- + addition
- - subtraction
- * multiplication
- / division
- % modulus (remainder)
- // floor division (rounds down)
- ** exponent (power)

Datatypes I struggled on:
- [ , , , ] = list (ordered, can store different data types)
- {x:y} = dictionary (key-value pairs, unordered)
- ( , , , ) = tuple (ordered, immutable)
- { , , , } = set (unordered, stores only unique items)

Euclidean distance formula = ((x2 - x1) ** 2 + (y2 - y1) ** 2) ** 0.5
